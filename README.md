# UfoChatcherGame
2019年高専祭ゲーム展示

## Description
学科展示で展示します．

## Install
* Unity 2019.2.5f1

## Directory
Assets直下にはプロジェクトである「Project」とそれ以外のフォルダが置かれます．AssetStore等からダウンロードしたものはインポートしたそのままの状態（Assets直下）にします．

Project以下にはファイルの種類によってフォルダ分けをします．
* Fonts：フォント
* Images：画像
* Prefabs：プレハブ
* Scenes：シーン
* Scripts：C#ソース
* ほか：必要に応じてAnimations,Models,Materialsなど
